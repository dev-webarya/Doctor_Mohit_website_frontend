

# Care and Cure Centre - Medical Clinic Website

## Overview
A professional, mobile-first medical website for Care and Cure Centre in Bangalore, featuring Dr. Mohit Singhal (Pediatrics & Neonatology) and Dr. Himanshu Singhal (Fertility Specialist). The design will be clean, trustworthy, and optimized for patient conversions.

---

## Design Direction
- **Style**: Clean, modern, and warm - perfect for families seeking pediatric and fertility care
- **Colors**: Soft blue and white with warm accent colors (trust-building, medical, approachable)
- **Typography**: Professional, easy-to-read fonts optimized for all ages
- **Imagery**: Placeholder spaces for clinic photos, doctor portraits, and medical icons

---

## Technical Requirements

### Mobile-First & Performance
- Fully responsive design (iPhone, Android, Tablet, Desktop)
- Optimized for fast loading (<3 seconds target)
- Lazy loading for images
- Clean, minimal code

### SEO Implementation
- Proper page titles and meta descriptions
- H1/H2 heading structure
- Medical clinic schema markup (LocalBusiness/Physician)
- Open Graph tags for social sharing
- Semantic HTML structure

### Lead Generation Features
- Floating WhatsApp button (always visible)
- Sticky header with "Book Now" CTA
- Multiple call-to-action points throughout
- Mobile-friendly phone buttons
